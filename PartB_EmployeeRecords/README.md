# Employee Records Servlet

This folder contains a Java Servlet that connects to a MySQL database using JDBC and displays employee records.

### Features
- Shows all employees in an HTML table
- Allows searching by Employee ID
- Demonstrates Servlet–JDBC integration

### Files
- `EmployeeServlet.java` — servlet code
- `db.sql` — SQL script to create sample table

### Note
This is demonstration code for GitHub submission only.  
If you want to actually run it, configure MySQL at:
`jdbc:mysql://localhost:3306/company` with user `root` and password `password`.
